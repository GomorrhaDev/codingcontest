### Wann sind die Kameras aus?
Aus irgendeinem unerklärlichen Grund hat das Matheinstitut erstaunlich viele 
Sicherheitskameras. Sie werden automatisch nach in festen Zeitintervallen neugestartet,
dadurch sind sie für ein paar Minuten nicht mehr funktionsfähig. Ihr habt eine Liste, auf 
der alle diese Zeitintervalle in Minuten notiert sind. Findet den frühstmöglichen Zeitpunkt,
an dem alle Kameras gleichzeitig neugestartet werden. 


### Beispiel 1
#### Input
```
2
3
4
```
#### Output
```
12
```

### Beispiel 2
#### Input
```
9846557
1596393
8426
30055839
```
#### Output
```
120631460156824046803758
```
